# Etheria
Etheria is a hotel booking platform built with React and Tailwind CSS that focuses on creating a smooth, interactive browsing experience. The interface adapts to each hotel’s visual style and responds instantly to user actions, making the booking flow more intuitive and engaging.

## Features
#### Mood Engine
- Each hotel is associated with a mood (e.g., luxury, cozy, modern). The UI updates dynamically—colors, typography, and styling adjust to match the hotel’s atmosphere.

#### Live Pricing
- The total price updates in real time as users add or remove services such as meals or amenities. This provides immediate and clear feedback during booking.

#### Cinematic Transitions
- Page transitions and UI interactions are smooth and consistent, improving flow and reducing abrupt context switches.

#### Responsive Design
- The layout is optimized for different screen sizes using Tailwind CSS, ensuring a consistent experience across devices.
